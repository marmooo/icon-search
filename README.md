# icon-search

[Icon Search](https://marmooo.github.io/icon-search/). 170+ icon sets with
280,000+ icons.

## Installation

- install [marmooo/icon-db](https://github.com/marmooo/icon-db)
- install [marmooo/rare-icon-db](https://github.com/marmooo/rare-icon-db)

## Build

```
bash build.sh
```
